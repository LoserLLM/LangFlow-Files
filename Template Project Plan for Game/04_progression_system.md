# Progression System

## Upgrade Tiers

1. **Faster Printers** ($200)
   - [ ] Increases printer speed by 20%
   - [ ] Reduces production time for all jobs

2. **Auto-Refill System** ($300)
   - [ ] Workers automatically refill printers from inventory
   - [ ] Eliminates manual filament management
   - [ ] Reduces downtime between print jobs

3. **Hire Worker** ($100)
   - [ ] Adds a new worker to automate tasks
   - [ ] Each additional worker increases overall efficiency
   - [ ] Workers can be hired multiple times for increased capacity

4. **Better Filament Capacity** ($150)
   - [ ] Increases maximum filament capacity by 200g
   - [ ] Reduces frequency of filament refills
   - [ ] Enables longer production runs without interruption

5. **Advanced Printers** ($500)
   - [ ] Increases maximum filament capacity by 50%
   - [ ] Increases printer speed by 50%
   - [ ] Higher quality output with fewer defects

6. **Conveyor Belts** ($800)
   - [ ] Automatically transports completed prints to delivery counter
   - [ ] Eliminates need for manual collection
   - [ ] Reduces worker travel time significantly

7. **AI Workers** ($1200)
   - [ ] Workers can predict job completion times
   - [ ] Position themselves optimally for immediate collection
   - [ ] Reduce idle time and improve overall workflow efficiency

8. **Multi-Level Factory** ($2000)
   - [ ] Increases filament station capacity by 50%
   - [ ] Unlocks new printer types with specialized capabilities
   - [ ] Enables parallel production lines for increased throughput
