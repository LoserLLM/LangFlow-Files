# Core Gameplay

## Character Control
- [ ] Move character using arrow keys to interact with printers and stations
- [ ] Space bar to interact with objects (printers, stations)

## Worker Automation
- [ ] Hire workers that use A* pathfinding to:
  - [ ] Collect completed prints
  - [ ] Refill filament from inventory
  - [ ] Deliver items to designated stations
- [ ] Workers intelligently navigate around obstacles

## Printer Management
- [ ] Start jobs on printers with varying speeds and capacities
- [ ] Manage filament inventory levels
- [ ] Repair broken machines through direct interaction
- [ ] Queue multiple jobs for continuous operation

## Progression System
- [ ] Earn credits by successfully delivering prints
- [ ] Use credits to purchase upgrades that enhance efficiency
- [ ] Each upgrade provides measurable improvements to production capabilities
